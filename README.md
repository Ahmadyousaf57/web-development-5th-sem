This repository contains **study material, exercises, and projects** from the **Web Development** course of the 5th semester.  
The focus of this course is to build **dynamic and responsive websites** using **HTML, CSS, Bootstrap, JavaScript, and PHP**.

---

# Topics Covered

# Frontend Development
- **HTML:** Structure, forms, tables, semantic tags  
- **CSS:** Styling, Flexbox, Grid, animations  
- **Bootstrap:** Responsive design, components, layout  
- **JavaScript:** DOM manipulation, events, validations, dynamic content

# Backend Development
- **PHP:** Server-side scripting, forms handling, session management  
- **Database Integration:** MySQL CRUD operations, connection with PHP  
- **Project Examples:** Login systems, small dynamic websites

# Practical Applications
- Creating responsive webpages  
- Building mini-projects (e.g., online store frontend, dashboards)  
- Integrating frontend with backend using PHP and MySQL  

---

# Technologies Used
- **Frontend:** HTML, CSS, Bootstrap, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL / SQLite  
- **Tools:** Visual Studio Code, XAMPP/WAMP

---

# Learning Outcomes
- Understand **frontend and backend web development**  
- Build **responsive and dynamic websites**  
- Apply **PHP & MySQL integration**  
- Gain hands-on experience in **DOM manipulation, forms, and layouts**
